Making a Claim

A certificate is uploaded by the user. This can be a scanned copy or link to view the certificate. The user can also add a link to his projects and other relevant information.
The user can get his skill verified by making a claim. The user has to spend E - tokens for getting the skill verified. When the user wants to validate the skill, validate button can be clicked. This triggers the On Chain Module. Based on the outcome of validation, ERP is awarded with.

Rating bias

The tendency of to rate someone or something differs from person to person. This tendency of scoring can be a result of various personality factors. The emotional quotient analysis on the portal can be used in understanding the attitude and personality. These results can be mapped with the scoring pattern (deviations from the mean) of the validator in order to calculate a bias which is added to the rating a validator is submitting for the claimant of the selected skill.
This bias can be helpful for the validator to minimise the deviation from the mean and thereby gain profit in form of EduCoin.

Easy to Use!

Educoin that uses the concept of giving the control of their data back to the users, reward them for sharing their skills and using the platform. This idea is based on the Ethereum blockchain, a framework in which individuals can make peer-to-peer transactions without needing to trust a third party or one another. Dapps work on blockchain 3.0, which implies that the backend code is running on a decentralised peer-to-peer network, a blockchain using a Proof of Stake concept. In Educoin, working professionals can ensure that their skills are validated by an unbiased majority, called validators. Validators whose responses get accepted to the blockchain are able to earn EduCoin, which can be utilised for making different purchases on the site. We make use of smart contracts for executing the transactions. The platform makes use of one tradable token (EduCoin) and one internal accounting token (ERP) that serves as an evaluation of the users on the respective skill sets. This rating can be useful but not limited in evaluation process by educational institutes as well as for professional grading of employees.
